# Infinity Quest

An AI-narrated fantasy RPG. Every turn is generated by Google Gemini: dice rolls decide outcomes, the narrator respects stat gates and continuity, and your save lives entirely in your browser.

This is a **bring-your-own-key** app. You supply a Gemini API key; the game calls Gemini directly from your browser. No server, no telemetry, no account.

## Run locally

**Prerequisites:** Node.js 20+.

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) and paste a Gemini API key when prompted. Get a free key at [aistudio.google.com/apikey](https://aistudio.google.com/apikey).

## Scripts

| Command | What it does |
| --- | --- |
| `npm run dev` | Start the Vite dev server on port 3000. |

## Privacy

- Your API key is stored **only** in `localStorage` on your device. It never leaves your browser except when calling Gemini directly.
- Your save file is stored in `localStorage`. Clearing site data deletes it.
- The app ships with `build.sourcemap: false` — the production bundle does not expose source maps.
- All images rendered in the game are `data:` URIs returned by Gemini; external URLs are rejected by an allowlist.

## Features

- Class-based runs (Warrior, Mage, Rogue, Cleric, Traveler) with distinct starting loadouts.
- Hunger, thirst, HP, mana, cooldowns, statuses — all enforced client-side to keep the LLM honest.
- Skill checks, world rolls, merchants, factions, reputation, bosses, three-act structure.
- Persistent meta-progression (Soul Shards, Pantheon, Sanctuary upgrades, Legacy items).
- Text narration, TTS audio, live voice mode, on-demand scene / map / item / enemy imagery.
- Keyboard-friendly: Escape closes modals, space/enter skips the typewriter, OS reduced-motion is honored.

## Known limitations

- Image quality and pacing depend on Gemini quota and latency. The app caches images client-side to minimize repeat calls.
- Saves are local only — clearing browser storage will lose your run.
- No cloud sync, no multiplayer.
